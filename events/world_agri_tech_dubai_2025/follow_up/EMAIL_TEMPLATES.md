## EMAIL BEST PRACTICES

### Timing

**Priority A:**
- Send within 24-48 hours (while memory is fresh)
- Follow up with call if no response after 3-4 days

**Priority B:**
- Send within 5-7 days
- Follow up if no response after 2 weeks

**Priority C:**
- Send within 7-10 days
- Add to nurture sequence, don't chase aggressively

### Personalization Checklist

Before sending ANY template:

‚úÖ Replace all [placeholders] with specific details  
‚úÖ Reference specific conversation points from your notes  
‚úÖ Mention their name, company, role correctly  
‚úÖ Adjust timezone for call scheduling  
‚úÖ Attach relevant materials (not generic stuff)  
‚úÖ Proofread for typos and grammar  
‚úÖ Test all links (calendar, website, attachments)  
‚úÖ Send from correct email (professional, not gmail if possible)  

### Subject Line Tips

‚úÖ Keep it short (< 50 characters)  
‚úÖ Include "World Agri-Tech" or "Dubai" for context  
‚úÖ Be specific, not vague  
‚úÖ Avoid spam triggers: "Free," "Limited time," excessive caps or punctuation  

### Email Structure

1. **Greeting** (Dear [Name], not "Hi" for formal contacts)
2. **Context** (Where you met, what you discussed - specific!)
3. **Value** (What you're offering, why it matters to THEM)
4. **Call to Action** (Clear next step, make it easy)
5. **Close** (Professional sign-off)
6. **Signature** (Full contact info)

### Attachment Strategy

**Do:**
- 1-3 attachments maximum
- Small file sizes (< 5MB total)
- PDF format (universal, professional)
- Clear file names ("Mustadem_Company_Profile.pdf" not "doc1.pdf")

**Don't:**
- Generic marketing materials that aren't relevant
- Huge files (> 10MB)
- Word docs with editable content
- Multiple attachments if one summary doc would suffice
