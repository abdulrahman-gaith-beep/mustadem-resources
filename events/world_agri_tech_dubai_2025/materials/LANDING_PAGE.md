## Form Design Tips

### Keep It Short
- Maximum 10 fields visible
- Use logic jumps to show additional fields only if relevant
- Don't ask for information you won't use

### Make It Mobile-Friendly
- Large input fields
- Large buttons
- Easy to tap dropdowns
- No horizontal scrolling

### Reduce Friction
- Only mark truly essential fields as "required"
- Use dropdowns/radio buttons instead of text when possible (faster)
- Auto-format phone numbers if possible
- Pre-fill "World Agri-Tech Dubai" for source field

### Build Trust
- Add privacy statement: "Your information will not be shared with third parties"
- Professional design with your branding
- HTTPS secure connection (most platforms provide this)

## QR Code Generation

### How to Create QR Code

**Step 1: Get Your Final URL**
- After building landing page/form, get the public URL
- Example: `https://mustadem.typeform.com/worldagritech2025`
- OR use url shortener: `bit.ly/mustadem-wat2025`

**Step 2: Generate QR Code**

**Free QR Code Generators:**
- qr-code-generator.com
- qrcode-monkey.com
- Canva (has built-in QR generator)

**Settings:**
- Type: URL
- Error correction: Medium or High
- Size: Minimum 300×300px for print (larger is better)
- Format: PNG or SVG
- Color: Black on white (high contrast, most reliable)

**Step 3: Test QR Code**
- Scan with your phone camera
- Test with iPhone AND Android
- Test in different lighting
- Verify it goes to correct page

**Step 4: Create Printable QR Signs**
- Large version for banners (8-10cm square minimum)
- Medium version for flyers (5-6cm square)
- Poster version if making standalone sign (A4 size with instructions)
