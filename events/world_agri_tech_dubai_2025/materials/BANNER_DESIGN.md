## Design Tips

### Colors
**Primary Palette:**
- Deep Blue (#003366) - Professional, water, trust
- Teal/Aqua (#00A9A5) - Technology, innovation, water
- White (#FFFFFF) - Clean, modern
- Gold/Copper accent (#D4AF37) - Premium, GCC aesthetic

**Background:**
- Clean white or very light blue gradient
- OR high-quality image with dark overlay for white text

### Typography
**Headlines**: Bold, sans-serif (e.g., Montserrat Bold, Gotham Bold)  
**Body Text**: Clean sans-serif (e.g., Open Sans, Helvetica)  
**Keep it LARGE** - people read banners from 3-5 meters away

### Images
- **HIGH QUALITY ONLY** (no pixelation at 200cm height)
- Professional RAS facility photos
- Modern, clean, high-tech aesthetic
- Avoid cluttered or low-quality images

### QR Code
- Minimum 8cm Ã— 8cm
- High contrast (black on white)
- Test before printing!
- Include short URL below QR as backup

## Content Hierarchy (Size Guide)

1. **Largest**: Main headline (10-15cm height text)
2. **Large**: Subheadline and benefits (5-8cm)
3. **Medium**: Body text and details (3-5cm)
4. **Small**: Website, taglines, fine print (2-3cm)

## Pre-Flight Checklist

Before sending to printer:

- [ ] Spell-check ALL text (get 3 people to review)
- [ ] QR code tested and works
- [ ] Logo is high-resolution (not blurry)
- [ ] Phone numbers and URLs are correct
- [ ] Color mode is CMYK (not RGB)
- [ ] File meets printer's size and bleed specs
- [ ] Saved as PDF with fonts embedded
- [ ] Preview at actual size if possible
- [ ] Get final approval from team lead
