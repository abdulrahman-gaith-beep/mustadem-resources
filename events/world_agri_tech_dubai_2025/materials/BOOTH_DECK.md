## Design Guidelines

### General Style
- **Clean, professional, modern**
- **High contrast** (dark text on light backgrounds or vice versa)
- **Consistent brand colors** throughout
- **Minimal text** per slide (designed to be read from 2-3 meters)
- **High-quality images** only (no pixelation, no stock photo watermarks)

### Color Scheme
**Primary:**
- Deep Blue (#003366)
- Teal/Aqua (#00A9A5)
- White (#FFFFFF)

**Accent:**
- Gold/Copper (#D4AF37)

**Backgrounds:**
- Solid white or light blue gradient
- OR high-quality image with dark overlay for white text
- Alternate between light and image backgrounds for variety

### Typography
**Slide Titles**: 48-60pt, Bold  
**Main Content**: 28-36pt  
**Supporting Text**: 20-24pt  
**Fine Print (URLs, etc.)**: 18-20pt

**Font**: Clean sans-serif (Montserrat, Gotham, Open Sans, or Helvetica)

### Images
- **Professional RAS facilities** (modern, clean, high-tech)
- **Fish in tanks** (healthy, clear water)
- **Team/operations shots** (if available)
- **Diagrams/infographics** (simple, clear, high-contrast)
- **NO low-resolution images**
- **NO generic stock photos** (smiling people pointing at screens, etc.)

### Icons
- Use consistent icon style throughout (line icons OR solid icons, not mixed)
- Large enough to be visible (5-8cm on slide)
- Simple, professional, modern

---

## Animation Settings

### Slide Transitions
- **Style**: Simple fade or none
- **Duration**: 0.5 seconds
- **Auto-advance**: 30-40 seconds per slide (adjust per slide as noted above)

### Content Animation
- **Minimal or none** (content loads fast, no distracting animations)
- If used: Simple fade-in only, fast timing
- **No flying text, spinning logos, or bouncing elements**

## Technical Setup

### PowerPoint Settings
1. Set to loop continuously:
   - Slideshow ‚Üí Set Up Slide Show ‚Üí Loop continuously until 'Esc'
2. Auto-advance timing set for each slide
3. Disable mouse click advancement (optional, for safety)

### PDF Alternative
- Export as PDF
- Use PDF reader in full-screen presentation mode
- Set to auto-advance if software allows (Adobe Acrobat Pro)

### Display Settings
- **Resolution**: 1920√ó1080 (Full HD) minimum
- **Aspect Ratio**: 16:9
- **Laptop screen brightness**: Maximum
- **Position laptop**: Angled slightly up, visible from front
