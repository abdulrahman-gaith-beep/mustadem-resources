# Finance Department | قسم المالية

## Department Overview | نظرة عامة

**English**: Manages all financial aspects including budgeting, cost tracking, financial modeling, and reporting.

**العربية**: يدير جميع الجوانب المالية بما في ذلك إعداد الميزانية وتتبع التكاليف والنمذجة المالية وإعداد التقارير.

---

## Workflow | سير العمل

### 1. Budgeting | إعداد الميزانية
- **Files**: `/budgeting/`
- Annual budget templates
- Department budget allocations
- Capital expenditure (CAPEX) planning
- Operating expenditure (OPEX) planning
- Cash flow projections

### 2. Financial Models | النماذج المالية
- **Files**: `/financial_models/`
- Project feasibility models
- ROI calculators
- Sensitivity analysis
- Break-even analysis
- 5-year projections

### 3. Cost Analysis | تحليل التكاليف
- **Files**: `/cost_analysis/`
- Cost per kg production
- Feed cost tracking
- Labor cost analysis
- Utility cost monitoring
- Profitability by project

### 4. Invoicing & AR | الفواتير والحسابات المدينة
- **Files**: `/invoicing/`
- Invoice templates
- Payment terms
- Accounts receivable tracking
- Collection procedures

### 5. Financial Reporting | التقارير المالية
- **Files**: `/reporting/`
- Monthly financial statements
- Quarterly reports
- Annual reports
- KPI dashboards
- Variance analysis

---

## Key Reports | التقارير الرئيسية

**Monthly**:
- P&L Statement
- Cash flow statement
- Budget vs Actual
- AR/AP aging

**Quarterly**:
- Executive summary
- Project profitability
- Forecast updates

**Annual**:
- Audited financials
- Tax filings
- Strategic planning

---

## Integration Points | نقاط التكامل

- **Sales**: Contract values, payment schedules
- **Construction**: Project costs, invoicing milestones
- **Production**: Operating costs, production metrics
- **Investment**: Financial projections, returns

**Training Duration**: 2-4 months (finance background required)
