# Construction Phase Checklist

## Overview

This checklist covers all activities required during the construction phase of a RAS (Recirculating Aquaculture System) project. It ensures quality construction, safety compliance, and project delivery.

---

## Pre-Phase Requirements

Before starting construction, confirm the following prerequisites are complete:

### Design Completion
- [ ] All design documents issued for construction
- [ ] Drawing list finalized and distributed
- [ ] Specifications complete
- [ ] Equipment datasheets finalized
- [ ] Design approved by owner

### Permitting and Approvals
- [ ] Building permit obtained
- [ ] Aquaculture license obtained or in process
- [ ] Environmental approvals received
- [ ] Utility connection approvals obtained
- [ ] Any other required permits secured

### Procurement
- [ ] Long-lead equipment ordered
- [ ] Delivery schedules confirmed
- [ ] Vendor drawings approved
- [ ] Shipping and logistics planned

### Contracting
- [ ] Construction contractor selected
- [ ] Contract executed
- [ ] Insurance requirements met
- [ ] Bonds in place (if required)
- [ ] Subcontractor approvals complete

### Site Preparation
- [ ] Site access available
- [ ] Temporary facilities planned
- [ ] Utilities for construction available
- [ ] Safety requirements communicated

---

## Construction Activities Checklist

### 1. Mobilization and Site Setup

**Contractor Mobilization**
- [ ] Contractor mobilized to site
- [ ] Construction management team in place
- [ ] Site office established
- [ ] Tool and equipment storage set up
- [ ] Material laydown areas designated

**Site Facilities**
- [ ] Temporary power connected
- [ ] Temporary water supply established
- [ ] Sanitary facilities installed
- [ ] First aid station established
- [ ] Fire extinguishers placed

**Safety Setup**
- [ ] Safety perimeter established
- [ ] Safety signage installed
- [ ] PPE requirements communicated
- [ ] Emergency procedures established
- [ ] Safety orientation conducted for all workers

**Documentation**
- [ ] Document control system active
- [ ] Daily log procedure established
- [ ] Progress reporting format agreed
- [ ] RFI (Request for Information) process defined
- [ ] Change order process defined

---

### 2. Site Work and Earthwork

**Site Clearance**
- [ ] Existing structures removed (if applicable)
- [ ] Vegetation cleared
- [ ] Debris removed
- [ ] Site surveyed and staked

**Earthwork**
- [ ] Excavation completed per drawings
- [ ] Fill and compaction completed
- [ ] Compaction tests passed
- [ ] Subgrade prepared
- [ ] Drainage provisions installed

**Underground Utilities**
- [ ] Utility trenches excavated
- [ ] Underground pipes installed
- [ ] Underground electrical conduits installed
- [ ] Utility connections made
- [ ] Trenches backfilled and compacted

**Inspection Points**
- [ ] Excavation depths verified
- [ ] Compaction tests documented
- [ ] Underground utilities inspected before backfill
- [ ] As-built documentation updated

---

### 3. Foundation and Concrete Work

**Foundation Preparation**
- [ ] Foundation excavation complete
- [ ] Subgrade inspected and approved
- [ ] Formwork installed
- [ ] Reinforcing steel placed per drawings
- [ ] Rebar inspection passed

**Concrete Placement**
- [ ] Concrete mix design approved
- [ ] Concrete delivery scheduled
- [ ] Pour sequence planned
- [ ] Concrete placed per specifications
- [ ] Curing procedures followed

**Concrete Testing**
- [ ] Slump tests performed
- [ ] Concrete cylinders taken for testing
- [ ] Compressive strength tests passed
- [ ] Test results documented

**Tank Foundations**
- [ ] Tank foundation locations verified
- [ ] Tank pad concrete placed
- [ ] Anchor bolts installed correctly
- [ ] Drain penetrations installed
- [ ] Waterproofing applied (if required)

**Inspection Points**
- [ ] Rebar placement inspected before pour
- [ ] Concrete placement observed
- [ ] Finished elevations verified
- [ ] Anchor bolt locations verified
- [ ] Concrete test results reviewed

---

### 4. Structural Steel and Building

**Steel Erection**
- [ ] Steel fabrication drawings approved
- [ ] Steel delivered to site
- [ ] Steel inspected for damage
- [ ] Anchor bolts checked for alignment
- [ ] Steel erected per erection drawings
- [ ] Bolted connections torqued and inspected
- [ ] Welded connections inspected (if applicable)

**Building Enclosure**
- [ ] Roof panels installed
- [ ] Wall panels installed
- [ ] Doors and windows installed
- [ ] Building weathertight
- [ ] Insulation installed

**Interior Construction**
- [ ] Interior partitions framed
- [ ] Wall finishes applied
- [ ] Floor coatings applied
- [ ] Ceiling systems installed
- [ ] Interior doors installed

**Inspection Points**
- [ ] Steel connections inspected
- [ ] Building envelope inspected for weather tightness
- [ ] Fire-rated assemblies verified
- [ ] Interior finishes quality checked

---

### 5. Tank Installation

**Tank Delivery and Placement**
- [ ] Tanks inspected upon delivery
- [ ] Tank placement coordinated with crane/rigging
- [ ] Tanks positioned per layout drawings
- [ ] Tanks leveled and secured
- [ ] Tank supports/stands installed

**Tank Connections**
- [ ] Drain connections made
- [ ] Inlet connections made
- [ ] Overflow connections made
- [ ] All connections watertight

**Tank Testing**
- [ ] Hydrostatic test performed (fill with water, 24-48 hours)
- [ ] Leak check completed
- [ ] Tank drains verified functional
- [ ] Test results documented

**Inspection Points**
- [ ] Tank condition upon delivery
- [ ] Tank placement and level
- [ ] Connection quality
- [ ] Hydrostatic test results

---

### 6. Mechanical Installation

**RAS Equipment Installation**
- [ ] Biofilter tanks/vessels installed
- [ ] Drum filter installed and leveled
- [ ] Pumps installed on bases
- [ ] Pump alignment checked
- [ ] Blowers installed
- [ ] Oxygen system installed
- [ ] UV sterilizers installed
- [ ] Heaters/chillers installed

**Piping Installation**
- [ ] Pipe supports installed
- [ ] Main circulation piping installed
- [ ] Drain piping installed
- [ ] Air piping installed
- [ ] Oxygen piping installed
- [ ] Domestic water piping installed
- [ ] Waste piping installed

**Piping Testing**
- [ ] Pressure tests performed
- [ ] Leak checks completed
- [ ] Test results documented
- [ ] System flushed

**HVAC Installation**
- [ ] HVAC units installed
- [ ] Ductwork installed
- [ ] Diffusers and grilles installed
- [ ] Insulation installed
- [ ] Refrigerant charged
- [ ] System tested

**Inspection Points**
- [ ] Equipment mounting verified
- [ ] Pump alignment checked
- [ ] Piping pressure tests witnessed
- [ ] HVAC system tested

---

### 7. Electrical Installation

**Main Distribution**
- [ ] Transformer installed (if applicable)
- [ ] Main distribution panel installed
- [ ] Sub-panels installed
- [ ] Cable trays/conduits installed
- [ ] Main cables pulled and terminated

**Power Distribution**
- [ ] Motor connections made
- [ ] VFDs installed and wired
- [ ] Control panels installed
- [ ] Receptacles installed
- [ ] Equipment power connected

**Lighting**
- [ ] Light fixtures installed
- [ ] Emergency lighting installed
- [ ] Exit signs installed
- [ ] Exterior lighting installed
- [ ] Lighting controls installed

**Emergency Power**
- [ ] Generator installed and connected
- [ ] Automatic transfer switch installed
- [ ] UPS system installed
- [ ] Emergency circuits verified

**Electrical Testing**
- [ ] Continuity tests performed
- [ ] Insulation resistance tests performed
- [ ] Ground tests performed
- [ ] Circuit identification completed
- [ ] Test results documented

**Inspection Points**
- [ ] Electrical rough-in inspected
- [ ] Panel installations inspected
- [ ] Generator tested
- [ ] Final electrical inspection passed

**Sensor Installation**

**Control System Installation**

- [ ] Control panel wiring completed
- [ ] I/O verified
- [ ] SCADA hardware installed
- [ ] Network infrastructure installed

**Alarm System**
- [ ] Alarm panels installed
- [ ] Alarm notification system installed
- [ ] Alarms tested
- [ ] Remote notification verified

**Inspection Points**
- [ ] Sensor locations verified per drawings
- [ ] Control panel wiring inspected
- [ ] Alarm functionality tested
- [ ] SCADA communication verified

---

### 9. Systems Integration

**Mechanical Systems**
- [ ] All piping connected
- [ ] Valves installed and tagged
- [ ] System filled with water
- [ ] Leaks repaired
- [ ] Flow verified through all loops

**Electrical Systems**
- [ ] All equipment energized
- [ ] Motor rotation verified
- [ ] VFD programming complete
- [ ] Emergency power tested

**Control Systems**
- [ ] All I/O verified
- [ ] Control sequences tested
- [ ] Setpoints programmed
- [ ] SCADA screens verified
- [ ] Alarm functions tested

**Integration Testing**
- [ ] Full system operation tested
- [ ] All interlocks verified
- [ ] Emergency shutdown tested
- [ ] Performance documented

---

### 10. Finalization and Punchlist

**Punchlist Development**
- [ ] Walkthrough conducted
- [ ] Punchlist items documented
- [ ] Items categorized by priority
- [ ] Responsibility assigned
- [ ] Completion dates established

**Punchlist Categories**
- [ ] Safety items (highest priority)
- [ ] Functional items (high priority)
- [ ] Cosmetic items (lower priority)
- [ ] Documentation items

**Punchlist Completion**
- [ ] Items addressed by contractor
- [ ] Items verified complete
- [ ] Re-inspection conducted
- [ ] Final punchlist cleared

**Site Cleanup**
- [ ] Construction debris removed
- [ ] Site cleaned and restored
- [ ] Temporary facilities removed
- [ ] Final grading completed
- [ ] Landscaping installed

## Quality Assurance and Inspections

### Required Inspections

| Inspection | Authority | Timing |
|------------|-----------|--------|
| Foundation/Rebar | Building dept/engineer | Before concrete pour |
| Concrete | Testing lab | During and after pour |
| Structural steel | Engineer/inspector | After erection |
| Underground utilities | Inspector | Before backfill |
| Electrical rough-in | Electrical inspector | Before covering |
| Plumbing | Plumbing inspector | Before covering |
| Fire suppression | Fire marshal | Before cover/final |
| Final building | Building dept | Before occupancy |
| Electrical final | Electrical inspector | Before energization |

- [ ] Daily logs maintained
- [ ] Progress photos taken

- [ ] All testing complete
- [ ] Punchlist items cleared (or plan for completion)

**Documentation Complete**
- [ ] As-built drawings issued
- [ ] Test reports complete
- [ ] Equipment documentation received
- [ ] Warranties in hand
- [ ] Certificates of completion issued

**Approvals Obtained**
